## About this org

This GitHub org is the **legal entity home** for QuantL Inc. Day-to-day engineering happens in [`QuantLinc`](https://github.com/QuantLinc); this org exists to hold:

- 📜 Corporate legal artifacts (data-room, MSAs, DPAs)
- 🔐 Customer-facing security disclosure ([security@quantl.click](mailto:security@quantl.click))

Customers, partners, or auditors: this is the right org to address.
