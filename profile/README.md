## About this org

This GitHub org is the **legal entity home** for QuantL Inc. Day-to-day engineering work happens in [`QuantLinc`](https://github.com/QuantLinc); this org exists to hold:

- 📜 Corporate legal artifacts (data-room, MSAs, DPAs)
- 🔐 Customer-facing security disclosure ([security@quantl.click](mailto:security@quantl.click))
- 📋 Audit-archive references (SOC 2 CC7.2 WORM bucket)

If you're a customer, partner, or auditor: this is the right org to address.
If you're a contributor or developer: head over to [`QuantLinc`](https://github.com/QuantLinc).
